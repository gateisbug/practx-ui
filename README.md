# Practics UI
실용적이고 범용적인 간단한 React 컴포넌트 라이브러리.

## Features
- fully support es6 Browser부터 지원
- es6 기반 Javascript 구성
- css variable 기반의 스타일링

## installation
```shell
# npm install practics-ui # use npm
# yarn add practics-ui # use yarn
```
